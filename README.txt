== Description ==

Show Products and Post as Instagram stories style, with button to go to the product page or post. Show them on any part of your site by adding the shortcode.

**Free Version Features:**

* Display stories from posts.
* Display unlimited stories by category.
* Create stories with only images.
* Order by Category.
* Order by Post

**Pro Version Features:**

* Create video stories.
* Create multiple stories.
* Integrate with Elementor and WP Bakery
* Styling options.
* More filter options

== Installation ==

1. Upload the entire `/bw-product-stories` directory to the `/wp-content/plugins/` directory.
2. Activate BW - Product Stories through the 'Plugins' menu in WordPress.
3. Use **[bw_show_stories]** shortcode in any wp-editor or Gutenberg.
4. Or insert **`<?php echo do_shortcode( '[bw_show_stories]' ); ?>`** code to .php file where you want to display.
5. Enjoy.
